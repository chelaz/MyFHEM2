Home Automation on your Wrist

Screenshots
Features
Overview
  • Send FHEM commannds
  • Receive status of FHEM devices
  • Show states with icons (on/off/toggle)
  • Speak commands by using the Dictation API (Pebble) for watches with microphone
Configuration
  • Request list of devices from FHEM server (can be filtered by given room eg. room Pebble)
  • Specifiy which devices should appear as favourites and/or used by dictation
  • Rename of room name, device name, states
  • Remember settings and reapply if new device list is received
Whats next?
Planned Features
  • Settings page on watch
  • Add custom states on configuration page
  • Default translation for built-in states: toggle/on/off
  • Sort favourites on configuration page
