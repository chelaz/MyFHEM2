WristFHEM - Pebble App for FHEM Home Automation Server

WristFHEM

Home Automation on your Wrist

This watch app is only working together with the FHEM Home Automation Server.

Please dontate to support and motivate the further development of this watch app by spending (for more details see Wikipedia’ Bitcoin article).

Screenshots
Features
Overview
  • Send FHEM commannds
  • Receive status of FHEM devices
  • Show states with icons (on/off/toggle)
  • Speak commands by using the Dictation API (Pebble) for watches with microphone
Configuration
  • Request list of devices from FHEM server (can be filtered by given room eg. room Pebble)
  • Specifiy which devices should appear as favourites and/or used by dictation
  • Rename of room name, device name, states
  • Remember settings and reapply if new device list is received

Documentation
Installation
  1. Download WatchApp
  2. Preferrably add the devices you want to see on the WatchApp to the room Pebble
  3. Go to settings page on phone and enter correct FHEM URL
  4. Start WatchApp and choose Request FHEM Devices in settings menu
  5. Go to settings page on phone. No you should see the list of your devices
  6. Check the first checkbox (favourites ) for all states of a device you want to see in the main menu.
  7. Check the second checkbox (dictation ) for all states of a device you want to use as a spoken command.
  8. Check the State of Device checkbox if you want to see the state of the device in the main menu.
  9. After pressing the submit button all the chosen devices should appear in the WatchApp
Usage
  • Special Menu: Dictate can be used to speak a command in the sequence ROOM ALIAS STATE
  • Favourites: By pressing the middle button the command and state to the chosen device is sent
  • States: Everytime you start the WatchApp all states are updated. If you press the middle button on a state it will be updated as well
